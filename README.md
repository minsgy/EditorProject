# 📕 에디터 프로젝트

VanilaJS를 활용하여 컴포넌트 방식의 에디터 작성하기 프로그램

- 자동 저장 및 편집, 하위 문서들을 관리 할 수 있는 '그' 프로그램과 비슷한 Editor 프로그램

## 프로젝트 기간

8월 28일 ~ 9월 3일, 7일

## 🛠 기술 스택

- HTML5
- CSS3
- JS(ES6)
- Storage(Local)
